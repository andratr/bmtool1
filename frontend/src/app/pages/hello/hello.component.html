Welcome to my project

Welcome to my project on benchmarking AI techniques for the use case of legacy systems modernization at ING.

Legacy codebases, often written in COBOL and PL/SQL, pose significant challenges for migration to modern platforms. This project evaluates and compares different AI-driven approaches to showcase how automation can accelerate and improve the process of code transformation.

This project builds on and is inspired by Leveraging LLMs for Automated Translation of Legacy Code: A Case Study on PL/SQL to Java Transformation (arXiv:2508.19663, 2025). I’m grateful for guidance and feedback from my supervisors— Andrea Capiluppi (university supervisor), Jonck van der Kogel (ING supervisor), and Zahra Moti (peer researcher / project peer), whose insights directly shaped the experimental design, evaluation criteria, and interpretation of results.

For methodology details, see: Solovyeva, L., Oliveira, E. C., Fan, S., Tuncay, A., Gareev, S., & Capiluppi, A. (2025). Leveraging LLMs for Automated Translation of Legacy Code: A Case Study on PL/SQL to Java Transformation. arXiv:2508.19663.
